## Flutter

::steps{level="3"}

### Ajouter la dépendance

```yaml [pubspec.yaml]
dependencies:
  qora_flutter: ^1.0.0
```

:::tip
`qora_flutter` inclut `qora` et `connectivity_plus` comme dépendances transitives. Aucun de ces packages ne nécessite d'entrée séparée dans `pubspec.yaml`.
:::

### Installer

```bash [Terminal]
flutter pub get
```

### Configuration de base

Créer un `QoraClient` et encapsuler la racine de l'application avec `QoraScope` :

```dart [main.dart]
import 'package:flutter/material.dart';
import 'package:qora_flutter/qora_flutter.dart';

void main() {
  final client = QoraClient(
    config: QoraClientConfig(
      defaultOptions: QoraOptions(
        staleTime: Duration(minutes: 5),
        cacheTime: Duration(minutes: 10),
        retryCount: 3,
      ),
      debugMode: true,
    ),
  );

  runApp(
    QoraScope(
      client: client,
      child: const MyApp(),
    ),
  );
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return const MaterialApp(home: HomeScreen());
  }
}
```

:::tip
Utiliser `debugMode: kDebugMode` en production afin que les logs de debug soient automatiquement supprimés.
:::
::

Passer à [Démarrage rapide](./quick-start.md) pour implémenter la première requête.

## Dart pur (sans Flutter)

Pour les outils CLI, les services backend ou les packages partagés sans Flutter, utiliser le package noyau directement :

```yaml [pubspec.yaml]
dependencies:
  qora: ^1.0.0
```

```bash
dart pub get
```

### Utilisation

```dart [main.dart]
import 'package:qora/qora.dart';

void main() async {
  final client = QoraClient(
    config: const QoraClientConfig(debugMode: true),
  );

  // Récupération ponctuelle avec cache et retry
  final user = await client.fetchQuery<User>(
    key: ['users', 1],
    fetcher: () => api.getUser(1),
  );

  print('User: ${user.name}');

  // Stream réactif : émet à chaque transition d'état
  client.watchQuery<User>(
    key: ['users', 1],
    fetcher: () => api.getUser(1),
  ).listen((state) {
    switch (state) {
      case Success(:final data):  print('Mis à jour : ${data.name}');
      case Failure(:final error): print('Erreur : $error');
      default: {}
    }
  });

  client.dispose();
}
```

## Optionnel : gestionnaires de cycle de vie et de connectivité

`FlutterLifecycleManager` invalide les requêtes quand l'application revient de l'arrière-plan. `FlutterConnectivityManager` invalide les requêtes quand la connexion réseau est rétablie. Les deux sont optionnels ; les omettre désactive le comportement de refetch automatique correspondant.

```dart [main.dart]
import 'package:qora_flutter/qora_flutter.dart';

void main() {
  final client = QoraClient();

  runApp(
    QoraScope(
      client: client,
      lifecycleManager: FlutterLifecycleManager(
        qoraClient: client,
        refetchInterval: const Duration(seconds: 30),
      ),
      connectivityManager: FlutterConnectivityManager(),
      child: const MyApp(),
    ),
  );
}
```

## Vérifier l'installation

```dart [verify.dart]
import 'package:qora/qora.dart';

void main() {
  final client = QoraClient();
  print('Qora installé avec succès.');
  client.dispose();
}
```

```bash
dart run verify.dart
# Qora installé avec succès.
```
