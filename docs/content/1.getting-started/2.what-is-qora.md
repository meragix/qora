**Qora** est une bibliothèque moderne de gestion d'état serveur pour Dart et Flutter, inspirée de [TanStack Query](https://tanstack.com/query) mais optimisée pour l'écosystème Dart/Flutter.

## Pourquoi Qora ?

La gestion d'état serveur est différente de la gestion d'état local. Qora vous aide à :

- **Synchroniser** vos données serveur avec votre UI
- **Mettre en cache** intelligemment pour améliorer les performances
- **Revalider** automatiquement les données périmées
- **Optimiser** les requêtes réseau avec la déduplication
- **Gérer** les erreurs et les retries de manière élégante

## Fonctionnalités principales

### Queries réactives

```dart
ReqryBuilder<User>(
  queryKey: QoraKey(['user', userId]),
  queryFn: () => api.getUser(userId),
  builder: (context, state) {
    return state.when(
      loading: (_) => CircularProgressIndicator(),
      success: (user, _) => Text(user.name),
      failure: (err, _, __) => Text('Error: $err'),
      initial: () => Text('No data'),
    );
  },
)
```

### Mutations optimistes

```dart
final mutation = client.mutation<User, UpdateUserInput>(
  mutationFn: (input) => api.updateUser(input),
  options: MutationOptions(
    onMutate: (ctx) async {
      // Mise à jour optimiste de l'UI
      final rollback = ctx.client.optimistic.update<User>(
        queryKey: QoraKey(['user', input.id]),
        updater: (old) => old?.copyWith(name: input.name),
      );
    },
  ),
);
```

### Pagination infinie

```dart
QoraInfiniteList<List<User>>(
  queryKey: QoraKey(['users', 'infinite']),
  queryFn: (page) => api.getUsers(page: page),
  itemBuilder: (context, user, index) => UserTile(user),
)
```

### Persistance du cache

```dart
final manager = client.enablePersistence(
  PersistenceConfig(
    persister: SharedPreferencesPersister(prefs),
    maxAge: Duration(days: 7),
  ),
);
```

## Architecture

Qora est conçu en deux packages complémentaires :

### `qora` - Core Dart pur

Package sans dépendance Flutter, utilisable dans n'importe quel projet Dart :

- ✅ CLI tools
- ✅ Backend Dart
- ✅ Scripts
- ✅ Tests unitaires

### `qora_flutter` - Intégration Flutter

Widgets et helpers Flutter pour une intégration transparente :

- ✅ `QoraScope` - InheritedWidget
- ✅ `QoraBuilder` - Widget réactif
- ✅ `QoraMutationBuilder` - Mutations avec UI
- ✅ `QoraInfiniteBuilder` - Pagination infinie
- ✅ Extension `context.qora`

## Philosophie de design

### Déclaratif, pas impératif

Qora favorise une approche déclarative où vous décrivez *quoi* afficher pour chaque état, plutôt que *comment* gérer les transitions d'état.

### Type-safe par défaut

Utilisation extensive des generics Dart et des sealed classes pour garantir la sûreté des types à la compilation.

### Flexible mais opinionné

Qora fournit des valeurs par défaut intelligentes tout en permettant une personnalisation complète pour les cas avancés.

### Performance d'abord

- Cache intelligent avec éviction automatique
- Déduplication des requêtes identiques
- Stale-while-revalidate pour une UX fluide
- Debouncing de la persistance

## Cas d'usage idéaux

Qora excelle dans :

- **Applications CRUD** - Gestion d'utilisateurs, produits, etc.
- **Feeds sociaux** - Pagination infinie, likes, commentaires
- **Dashboards** - Données temps réel avec revalidation
- **Apps offline-first** - Persistance + optimistic updates
- **E-commerce** - Panier, wishlist, commandes
