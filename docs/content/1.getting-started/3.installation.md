## Flutter

::steps{level="3"}

### Add the dependency

```yaml [pubspec.yaml]
dependencies:
  qora_flutter: ^1.0.0
```

:::tip
`qora_flutter` includes `qora` and `connectivity_plus` as transitive dependencies. Neither package requires a separate entry in `pubspec.yaml`.
:::

### Install

```bash [Terminal]
flutter pub get
```

### Basic setup

Create a `QoraClient` and wrap the application root with `QoraScope`:

```dart [main.dart]
import 'package:flutter/material.dart';
import 'package:qora_flutter/qora_flutter.dart';

void main() {
  final client = QoraClient(
    config: QoraClientConfig(
      defaultOptions: QoraOptions(
        staleTime: Duration(minutes: 5),
        cacheTime: Duration(minutes: 10),
        retryCount: 3,
      ),
      debugMode: true,
    ),
  );

  runApp(
    QoraScope(
      client: client,
      child: const MyApp(),
    ),
  );
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return const MaterialApp(home: HomeScreen());
  }
}
```

:::tip
Set `debugMode: kDebugMode` in production builds so debug logs are stripped automatically.
:::
::

## Pure Dart (no Flutter)

For CLI tools, backend services, or shared packages without Flutter, use the core package directly:

```yaml [pubspec.yaml]
dependencies:
  qora: ^1.0.0
```

```bash
dart pub get
```

### Usage

```dart [main.dart]
import 'package:qora/qora.dart';

void main() async {
  final client = QoraClient(
    config: const QoraClientConfig(debugMode: true),
  );

  // One-shot fetch with caching and retry
  final user = await client.fetchQuery<User>(
    key: ['users', 1],
    fetcher: () => api.getUser(1),
  );

  print('User: ${user.name}');

  // Reactive stream: emits every state transition
  client.watchQuery<User>(
    key: ['users', 1],
    fetcher: () => api.getUser(1),
  ).listen((state) {
    switch (state) {
      case Success(:final data):  print('Updated: ${data.name}');
      case Failure(:final error): print('Error: $error');
      default: {}
    }
  });

  client.dispose();
}
```

## Optional: Lifecycle and Connectivity Managers

To enable automatic refetch on app resume and refetch on network reconnect, provide the Flutter managers to `QoraScope`:

```dart [main.dart]
import 'package:qora_flutter/qora_flutter.dart';

void main() {
  final client = QoraClient();

  runApp(
    QoraScope(
      client: client,
      lifecycleManager: FlutterLifecycleManager(
        qoraClient: client,
        refetchInterval: const Duration(seconds: 30),
      ),
      connectivityManager: FlutterConnectivityManager(),
      child: const MyApp(),
    ),
  );
}
```

## Verify the Installation

```dart [verify.dart]
import 'package:qora/qora.dart';

void main() {
  final client = QoraClient();
  print('Qora installed successfully.');
  client.dispose();
}
```

```bash
dart run verify.dart
# Qora installed successfully.
```
