## Installation pour Flutter

### 1. Ajouter les dépendances

Ajoutez `flutter_qora` à votre `pubspec.yaml` :

```yaml [pubspec.yaml]
dependencies:
  flutter:
    sdk: flutter
  flutter_qora: ^0.1.0
```

:::tip
`flutter_qora` inclut automatiquement `qora` comme dépendance, vous n'avez pas besoin de l'ajouter séparément.
:::

### 2. Installer les packages

```bash [Terminal]
flutter pub get
```

### 3. Configuration de base

Créez un client Qora global dans votre `main.dart` :

```dart [main.dart]
import 'package:flutter/material.dart';
import 'package:flutter_qora/flutter_qora.dart';

void main() {
  // Créer le client Qora
  final qoraClient = QoraClient(
    config: QoraClientConfig(
      defaultOptions: QoraOptions(
        staleTime: Duration(seconds: 30),
        cacheTime: Duration(minutes: 5),
        retryCount: 3,
      ),
      debugMode: true, // Activer en développement
    ),
  );

  runApp(
    QoraScope(
      client: qoraClient,
      child: MyApp(),
    ),
  );
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Qora Demo',
      home: HomeScreen(),
    );
  }
}
```

✅ Vous êtes prêt ! Consultez le [Quick Start](./quick-start.md) pour créer votre première query.

## Installation pour Dart pur

Si vous travaillez sur un projet Dart sans Flutter (CLI, backend, etc.), utilisez uniquement le package core :

### 1. Ajouter la dépendance

```yaml [pubspec.yaml]
dependencies:
  qora: ^0.1.0
```

### 2. Installer

```bash [Terminal]
dart pub get
```

### 3. Utilisation

```dart [main.dart]
import 'package:qora/qora.dart';

void main() async {
  final client = QoraClient(
    config: QoraClientConfig(debugMode: true),
  );

  // Exécuter une query
  final user = await client.fetchQuery<User>(
    key: QoraKey(['user', 1]),
    fetcher: () => api.getUser(1),
  );

  print('User: ${user.name}');

  // Observer les changements
  client.watchState<User>(QoraKey(['user', 1])).listen((state) {
    state.when(
      success: (user, _) => print('Updated: ${user.name}'),
      error: (err, _, __) => print('Error: $err'),
      loading: (_) => print('Loading...'),
      initial: () => print('Initial'),
    );
  });

  // N'oubliez pas de disposer
  client.dispose();
}
```

## Dépendances optionnelles

### Pour la persistance (SharedPreferences)

```yaml [pubspec.yaml]
dependencies:
  flutter_qora: ^0.1.0
  shared_preferences: ^2.2.0
```

Puis configurez la persistance :

```dart [main.dart]
import 'package:shared_preferences/shared_preferences.dart';

void main() async {
  WidgetsFlutterBinding.ensureInitialized();
  
  final prefs = await SharedPreferences.getInstance();
  final persister = SharedPreferencesPersister(prefs);
  
  final client = QoraClient(
    config: QoraClientConfig(debugMode: true),
  );
  
  final persistenceManager = client.enablePersistence(
    PersistenceConfig(
      persister: persister,
      maxAge: Duration(days: 7),
    ),
  );
  
  runApp(
    QoraScope(
      client: client,
      child: MyApp(),
    ),
  );
}
```

### Pour le monitoring et les logs

```yaml [pubspec.yaml]
dependencies:
  flutter_qora: ^0.1.0
  logger: ^2.0.0
```

## Configuration pour un monorepo Melos

Si vous utilisez Melos pour gérer un monorepo :

### Structure du projet

```bash
my_app/
├── packages/
│   ├── mobile/          # App Flutter
│   ├── core/            # Logique métier
│   └── api/             # Client API
├── melos.yaml
└── pubspec.yaml
```

### `melos.yaml`

```yaml [melos.yaml]
name: my_app

packages:
  - packages/**

scripts:
  bootstrap:
    run: melos bootstrap
    description: Install all package dependencies
  
  analyze:
    run: melos exec -- flutter analyze
    description: Run static analysis
  
  test:
    run: melos exec -- flutter test
    description: Run all tests
```

### `packages/mobile/pubspec.yaml`

```yaml [pubspec.yaml]
name: mobile
description: Flutter mobile app

dependencies:
  flutter:
    sdk: flutter
  flutter_qora: ^0.1.0
  
  # Dépendances locales
  core:
    path: ../core
  api:
    path: ../api
```

### Bootstrap

```bash [Terminal]
melos bootstrap
```

## Vérification de l'installation

Créez un fichier `test_qora.dart` :

```dart [test_qora.dart]
import 'package:flutter_qora/flutter_qpra.dart';

void main() {
  final client = QoraClient();
  print('✅ Qora installé avec succès !');
  print('Version: 0.1.0');
  client.dispose();
}
```

Exécutez :

```bash [Terminal]
dart run test_qora.dart
```

Vous devriez voir :

```bash
✅ Qora installé avec succès !
Version: 0.1.0
```
